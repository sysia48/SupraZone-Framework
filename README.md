# SupraZone Framework

<a href="https://pypi.org/project/suprazone-framework/"><img src="https://img.shields.io/badge/pypi-placeholder-blue"></a> <a href="https://github.com/sysia48/SupraZone-Framework/actions"><img src="https://img.shields.io/badge/actions-placeholder-brightgreen"></a>

SupraZone Framework is a small, extensible Python library intended to provide helpers and abstractions for building command-line utilities and small services. This repository contains the core library, tests, and examples.

Quick install:

```bash
pip install suprazone-framework
```

Basic usage:

```python
from suprazone import example

print(example.hello())
```
